Front-end developer

I am a front-end developer located in Tallinn. My goal is to have a career as a front-end developer. As Skype had made Estonia stand out to the world, my dream is also creating a website or a service that would represent Estonia.
I find passion in writing a code and keeping it clean. Additionally, I get really excited about discovering elegant web designs and trying to code them on my own. Through one of projects in my previous workplace I have found a passion to keep a keen eye on UI/UX features.

WEB DEVELOPMENT

I find passion in writing a clean and reusable code and keeping it clean. I also like to maintain readability in order for other people to understand my code.

WEB DESIGN

While designing a website, I keep my mind on trending web designs in order to find an inspiration as well as customers' needs.

RESPONSIVE LAYOUT

My method for writing a code is "mobile first," so before I actually dive into coding, I think ahead and design a layout that is suitable for all platforms - mobile, tablets and desktop.

TOOLS

Technologies I love to use in my works: HTML5, CSS3 ja Jquery. Bootstrap for a framework, Adobe XD for web design and SCSS preprocessor.

UI/UX DESIGN

With coding works, I always feel there needs to be a good balance between user interface and user experience. The presentation of a web page has to be elegant and usable as well as user experience determines if the users are satisfied.

ACCESSIBILITY

I write HTML5 code with accessibility in mind so I keep my code as semantic as possible. For example, alternative text for an image or .blind class which includes information about each section in the code.
